# Frontend development utilities

| File | Purpose |
|------|---------|
| `check-imports.js` | Import path validation |
| `test-auth.js` | Auth flow smoke test (`node scripts/dev/test-auth.js`) |
| `test-signin.html` | Manual sign-in page test |
| `verify-task-pages.js` | Task page route checks |
| `test.pdf` | Sample PDF for upload tests |

Run from the `frontend/` directory:

```bash
node scripts/dev/test-auth.js
node scripts/dev/verify-task-pages.js
```
